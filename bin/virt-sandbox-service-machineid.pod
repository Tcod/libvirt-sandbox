=head1 NAME

virt-sandbox-service machineid - Show the /etc/machine-id specific to container

=head1 SYNOPSIS

  virt-sandbox-service [-c URI] machineid [-h] NAME

=head1 DESCRIPTION

virt-sandbox-service is used to manage secure sandboxed system services.
These applications will be launched via libvirt and run within a virtualization
technology such as LinuX Containers (LXC), or optionally QEMU/KVM. The
container / virtual machines will be secured by SELinux and resource
separated using cgroups.

The machineid command show the /etc/machine-id specific to sandboxed container

=head1 OPTIONS

=over 4

=item B<-h>, B<--help>

Display help message

=item B<-c> URI, B<--connect URI>

The connection URI for the hypervisor (only LXC or QEMU are
supported currently).

=back

=head1 EXAMPLE

Machineid the httpd1 container

 # virt-sandbox-service machineid httpd1
 1b16d5a8cec649e7ba7d9f9f6ef8f393

 You can use this command on the host with journalctl like the following.

 # journalctl --since today --merge _MACHINE_ID=`virt-sandbox-service machineid httpd1`
